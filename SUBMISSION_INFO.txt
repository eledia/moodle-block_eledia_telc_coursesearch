================================================================================
PLUGIN DESCRIPTION
================================================================================

The eLeDia TELC Course Search block provides advanced course search and
filtering functionality for Moodle, including:
- Multi-criteria search (categories, tags, custom fields)
- Multiple view modes (cards, list, summary)
- Progress tracking
- Favorite courses
- Responsive design
- Accessibility features
